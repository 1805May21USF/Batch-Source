JS Homework Part 1

1. Return the nth Fibonacci number.
Enter in console: homework.fibonacci(n) where n is nth place

2. Sort array of integers
Enter in console: homework.sort(array) where array is ex: [1,2,3,4].

3. Return the factorial of n
Enter in console: homework.factor(n) where n is the number needed to retrieve factorial.

4. Rotate left
Enter in console: homework.rotateLeft(array, n) where array is ex: [1,2,3,4] and n is the number of times to rotate.

5. Balanced Brackets Enter in console: homework.balancedBrackets(input) where input is ex: "{", "}", "{}", "(", ")", "()", "()()"
